# Sistema Web de Gestión Académica ACIS

## Proyecto de Gestión Académica
Este sistema integral permite la administración eficiente de procesos educativos, incluyendo la gestión de alumnos, matrículas, seguimiento académico y finanzas escolares.

## Resumen del Proyecto
El **Sistema ACIS** es una plataforma web desarrollada en Laravel diseñada para centralizar la operación académica. Sus módulos principales incluyen:
*   **Gestión de Alumnos:** Registro y seguimiento detallado de la información del estudiante.
*   **Gestión Académica:** Administración de cursos, horarios y rendimiento.
*   **Matrícula:** Procesos automatizados para la inscripción de alumnos.
*   **Finanzas:** Control de pagos, deudas y reportes financieros.

## Requisitos del Sistema
*   PHP >= 8.2
*   Composer
*   MySQL / MariaDB
*   Node.js & NPM

## Instalación Rápida
1. Clonar el repositorio.
2. Instalar dependencias: `composer install` y `npm install`.
3. Configurar `.env` (Base de datos y llaves).
4. Generar llave: `php artisan key:generate`.
5. Ejecutar migraciones: `php artisan migrate`.
6. Iniciar servidor: `php artisan serve`.
